## 🎬 Movie Ticket Booking Form
 
A simple and interactive **Movie Ticket Booking Form** built using **HTML** and **CSS** as a beginner web development project.
 
---
 
## 📋 Overview
 
This project is a front-end web form that allows users to book movie tickets by filling in their personal details, selecting a movie, choosing seats, and picking a payment method — all within a clean, styled single-page layout.
 
---
 
## ✨ Features
 
- **Customer Details** — Name, Email, and Phone Number fields
- **Movie Selection** — Choose from available movies (Kalki, Salaar, Radha Shyam, Bahubali)
- **Date & Show Time** — Pick preferred date and time
- **Theater Selection** — Choose from Hyderabad theaters (AMB Cinemas, PVR ICON, INOX, Prasads Multiplex)
- **Seat Type** — Regular, Premium, or VIP via radio buttons
- **Food & Beverages** — Optional add-ons: Popcorn, Soft Drink, Combo Meal
- **Payment Method** — UPI, Credit Card, Debit Card, or Net Banking
- **Form Reset & Submit** buttons

## 🚀 How to Run
 
1. **Clone the repository**
```bash
   git clone https://github.com/your-username/movie-ticket-booking.git
```
 
2. **Navigate to the project folder**
```bash
   cd movie-ticket-booking
```
 
3. **Open the file in your browser**
```bash
   # Just double-click the HTML file, or open it with:
   open Movie_Ticket_Booking_Form.html
```
 
No installations or dependencies needed — it's pure HTML & CSS!
